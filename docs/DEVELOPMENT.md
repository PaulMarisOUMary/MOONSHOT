# Development

## Requirements

- [Python3](https://www.python.org/downloads/)
- pip3

## Setup

```bash
# Install dependencies
pip3 install -r requirements.txt
```

## Install library

```bash
# Install library
pip3 install -U .
```

## Run tests

```bash
# Run tests
pytest
```

You can also use Visual Studio Code to run the tests in the Testing tab.
