# Development

## Requirements

- [Python 3](https://www.python.org/downloads/) (tested with Python 3.11.x)
- [pip](https://pip.pypa.io/en/stable/installation/) (tested with pip 23.2.x)

## Setup

```bash
# Install dependencies
pip3 install -r requirements.txt
```

## Install library

```bash
# Install library
make build
```

## Run tests

```bash
# Run tests
make tests
```
